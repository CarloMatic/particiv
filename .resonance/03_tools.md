# Tools & Boundaries – PartiCiv

## Allowed
- Next.js 15 (App Router)
- Vanilla CSS (no Tailwind)
- Lucide React (icons)
- Google Fonts (Inter, or similar professional sans-serif)
- Static site generation (SSG)

## Forbidden
- No Tailwind CSS
- No external CMS
- No database
- No client-side JavaScript frameworks beyond React
- No third-party analytics without DSGVO compliance

## Hosting
- Static export → German hosting provider or Vercel (EU region)
